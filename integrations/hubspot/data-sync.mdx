This page documents what data flows between VoiceLine and HubSpot, in which direction, and on what schedule.

## HubSpot naming note

HubSpot uses "Companies" instead of "Accounts" and "Deals" instead of "Opportunities". VoiceLine maps these internally — reps see "Account" and "Opportunity" labels in the app, which sync to Company and Deal objects in HubSpot.

## Sync overview

| Object | Direction | Trigger |
|--------|-----------|---------|
| Companies (Accounts) | HubSpot → VoiceLine | On login + scheduled refresh |
| Contacts | HubSpot → VoiceLine | On login + scheduled refresh |
| Deals (Opportunities) | HubSpot → VoiceLine | On login + scheduled refresh |
| Owners | HubSpot → VoiceLine | On login |
| Visit reports | VoiceLine → HubSpot Note | On submission |
| Call logs | VoiceLine → HubSpot Note | On submission |
| Meetings | VoiceLine → HubSpot Meeting | On submission |
| Tasks | VoiceLine → HubSpot Task | On submission |
| New contacts | VoiceLine → HubSpot Contact | On submission |
| Deal updates | VoiceLine → HubSpot Deal | On submission |

## Outbound: VoiceLine → HubSpot

### Visit report → HubSpot Note

Notes are associated with the relevant Company, Contact, and/or Deal based on the visit context.

| VoiceLine field | HubSpot property | Notes |
|-----------------|-----------------|-------|
| Visit summary | body | Full AI-generated summary |
| Account | associations → company | Linked via association |
| Contact | associations → contact | Linked via association |
| Visit date | timestamp | Date of submission |

## Sync cadence

| Direction | Trigger | Details |
|-----------|---------|---------|
| HubSpot → VoiceLine | On rep login | Full refresh of companies, contacts, deals |
| HubSpot → VoiceLine | Scheduled | Background refresh — interval configured during implementation (typically 4–6 hours) |
| VoiceLine → HubSpot | On submission | Immediate |

## What VoiceLine does not sync

- HubSpot Marketing Hub data (campaigns, email sequences)
- Service Hub tickets
- Custom HubSpot objects (unless configured as a non-standard implementation)

<Note>Field-level mappings shown here are the standard defaults. Your SA may configure additional mappings during implementation to match your HubSpot property names and pipeline stage values.</Note>
