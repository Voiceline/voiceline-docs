> **€900 / month**

Standard platform package for enterprise deployments. The right choice for organisations deploying VoiceLine in German and English, using OAuth or password authentication, and managing a single organisational hierarchy.

## Languages

| Capability | What it does |
|-----------|-------------|
| German and English workspace languages | Reps record in German or English; AI-generated reports are written in the configured output language |
| Report output language | Output language can differ from input language — reps can speak German and have reports written in English |
| Up to 3 assistant languages | Support up to three recording languages across your rep population |
| Product catalogue sync | Upload or sync your product catalogue so reps can reference and tag products during recordings |
| Competitor upload | Add competitors to the workspace so reps can tag them in market feedback and visit reports |
| Custom vocabulary | Add product aliases and internal terms to improve speech recognition accuracy for specialist terminology |

→ [Language Settings](/customer/platform/language-settings)

## Authentication

| Capability | What it does |
|-----------|-------------|
| Password authentication | Email and password login; user credentials managed in VoiceLine |
| OAuth 2.0 | Standard OAuth flow via your identity provider |
| Data processing consent | Display a GDPR recording consent prompt to reps before each session; consent timestamps are logged |

→ [Security](/customer/platform/security)
