Everything in Professional, plus expanded language support, SSO, and multi-function workspace partitioning. The right choice for large or complex organisations that require SSO, need to support languages beyond German and English, or need to partition the workspace into independent units with separate feature sets (for example, field sales and service technicians under one account but fully isolated).

## Languages

Everything in Professional, plus:

| Additional capability | What it does |
|----------------------|-------------|
| Custom workspace language | Deploy VoiceLine in any language beyond German and English — configured per workspace |
| Unlimited assistant languages | No cap on the number of recording languages across your rep population |
| Workflow translations | UI labels and workflow prompts are translated into the workspace language |
| Per-group language configuration | Set input and output language independently per team or rep — for multi-country deployments |
| CRM translation | Reports are translated on the fly from the rep's recording language to the workspace language when synced to the CRM |
