Platform packages control languages, authentication, and user management capabilities. One package is selected per workspace — it sits beneath Field Sales Fundamentals and applies to the entire workspace.

| Package | Price | Best for |
|---------|-------|---------|
| [Professional](/commercial/platform/professional) | €900 / month | Standard enterprise deployments in German or English, with OAuth or password auth and a single org hierarchy |
| [Enterprise](/commercial/platform/enterprise) | €2,490 / month | Organisations requiring SSO, languages beyond German and English, or workspace partitioning |

## What platform packages cover

**Languages** — the language reps record in, the language AI outputs (reports, call logs) are written in, and custom vocabulary for accurate speech recognition. Report output language can differ from input language: reps can speak German and receive English reports. Enterprise adds support for any language and per-group configuration for multi-country deployments.

**Authentication** — how users log in to VoiceLine. Professional supports password and OAuth. Enterprise adds SSO via corporate identity providers such as Azure AD or Okta. All tiers support data processing consent prompts for GDPR-regulated environments.

**User management** — user roles, invitation flows, and organisation structures that scope manager visibility. Enterprise additionally supports syncing permissions and org structures from an external source, and partitioning the workspace into independent units — each with its own feature set, workflows, and search logic.
