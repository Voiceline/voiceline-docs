VoiceLine is sold as a base package plus optional modules and a platform tier.

| Layer | What it is | Pricing |
|-------|-----------|---------|
| [Field Sales Fundamentals](/commercial/field-sales-fundamentals) | Base package — required for all customers | €39 / user / month |
| [Modules](/commercial/modules/index) | Add-on capabilities unlocked on top of FSF | From €150 / month |
| [Platform](/commercial/platform/index) | Infrastructure tier — languages, auth, user management | From €900 / month |

<Note>Modules are available only after Field Sales Fundamentals has been successfully rolled out. Implementation fees apply separately.</Note>

## Implementation fees

Custom implementation work is billed at **€1,300 per person day**. This applies to:
- VPN setup
- Non-standard CRM object implementation
- Non-standard CRM system integration
