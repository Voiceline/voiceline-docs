VoiceLine pricing is structured in four layers that stack on top of each other. Every customer starts with Field Sales Fundamentals and selects a Platform package. Modules are activated on top of that foundation once the rollout is complete.

| Layer | What it is | Billed |
|-------|-----------|--------|
| [Field Sales Fundamentals](/commercial/field-sales-fundamentals) | Base package — required for all customers. Covers frontline productivity, market analytics, and sales coaching. | €39 / user / month |
| [Platform](/commercial/platform/index) | Infrastructure tier — workspace languages, authentication, and user management. One package per workspace. | From €900 / month flat |
| [Modules](/commercial/modules/index) | Add-on capabilities that extend FSF with additional workflows or analytics depth. Activated after rollout. | From €150 / month flat |
| [Credits](/commercial/credits) | Usage-based layer — measures CRM records and insights produced. FSF includes a baseline allowance per user; packages for heavy usage. | 150 credits / user / month included; packages from €105 |

<Note>Module and Platform fees are flat per-workspace — they do not scale with seat count. Field Sales Fundamentals is per-user. Credits scale with actual product usage. Implementation fees are billed separately (see below).</Note>

## How the layers stack

A typical customer pays:

1. **FSF** — €39 × number of users per month
2. **One Platform package** — either Professional (€900/mo) or Enterprise (€2,490/mo) per workspace
3. **Any activated modules** — flat monthly fees added per module
4. **Credits** — 150 credits/user/month included in FSF; purchase packages when usage exceeds the baseline

For example: 30 users on Professional with Pipeline Management = (30 × €39) + €900 + €400 = **€2,470 / month** before credits.

## How ARR scales with a customer

Deeper product adoption drives ARR growth — not seat count alone.

| Customer journey | What changes |
|-----------------|-------------|
| FSF only | Base per-seat revenue; credits within baseline |
| + Platform Professional | +€900/month for workspace infrastructure |
| + modules activated | +€150–€900/month per module, each with its own value case |
| + heavy usage | Credit consumption exceeds baseline → packages add a usage-proportional revenue layer |

The credit layer means enterprise customers — who use VoiceLine for more interactions, more CRM records, and more analytics extractions — naturally pay more as they extract more value, without requiring separate pricing tiers. See [Pricing Architecture](/commercial/pricing-architecture) for the full strategic framework.

## Implementation fees

Custom implementation work is billed at **€1,300 per person day**. This applies to:

- VPN setup
- Non-standard CRM object implementation
- Non-standard CRM system integration
