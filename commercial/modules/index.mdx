Add-on modules that extend Field Sales Fundamentals with additional capabilities and use cases.

Each module activation adds a new revenue-bearing capability to the workspace with its own value case. The module flat fee covers the intrinsic value of having the capability available. Usage scales independently through [Credits](/commercial/credits) — as reps adopt a module and use it more, credit consumption grows naturally without changing the module price. This means low-adoption early stages are affordable, and high-adoption enterprise scale is correctly priced.

<Note>All modules require [Field Sales Fundamentals](/commercial/field-sales-fundamentals) to be fully rolled out before activation.</Note>
