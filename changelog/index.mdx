<Update label="2026-04-21" tags={["Improvement"]} rss={{ title: "Improved event selection accuracy and language support" }}>
  ## Improved event selection accuracy and language support

  Event selection — the step where you link a visit report or workflow to a CRM calendar event — now produces more accurate AI-generated fields. Titles, time ranges, and other event details are generated correctly across all supported languages.

  **What changed:**
  - AI-generated event titles and time ranges are now more accurate when creating or matching CRM events
  - Event fields are generated in your configured working language, fixing cases where multilingual workspaces saw incorrect or mixed-language output
  - Pre-planned visits with a linked calendar event are matched more reliably

  No action is required — these improvements apply automatically to all workspaces using event selection.
</Update>
