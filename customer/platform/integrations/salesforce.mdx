Connects VoiceLine to Salesforce via OAuth 2.0, syncing accounts, contacts, leads, and opportunities in both directions.

## Overview

VoiceLine pulls accounts, contacts, leads, opportunities, and campaigns from Salesforce so reps can search their full customer base from the field. When reps submit field activities, VoiceLine pushes structured records back to Salesforce — visit reports as events, call logs as phone call activities, and tasks as Salesforce tasks — linked to the relevant account or contact.

## Where to configure it

Go to **Workspace Settings** → **Integrations** → **Salesforce**.

## Setup

**Prerequisites:**
- Salesforce account with API access enabled
- A Connected App configured in Salesforce (for OAuth 2.0)
- Workspace administrator permissions in VoiceLine

**Steps:**
1. Open **Workspace Settings** → **Integrations**
2. Select **Salesforce**
3. Click **Connect with Salesforce** — you'll be redirected to Salesforce to authorize the connection
4. Log in with your Salesforce administrator credentials and approve access
5. You're redirected back to VoiceLine — the integration is active and the initial sync begins

## Sync behavior

| Object | Direction | When |
|--------|-----------|------|
| Accounts | Salesforce → VoiceLine | On login / scheduled refresh |
| Contacts | Salesforce → VoiceLine | On login / scheduled refresh |
| Leads | Salesforce → VoiceLine | On login / scheduled refresh |
| Opportunities | Salesforce → VoiceLine | On login / scheduled refresh |
| Campaigns | Salesforce → VoiceLine | On login / scheduled refresh |
| Visit reports | VoiceLine → Salesforce (Event) | On submission |
| Call logs | VoiceLine → Salesforce (Phone Call Activity) | On submission |
| Tasks | VoiceLine → Salesforce (Task) | On submission |
| Meetings | VoiceLine → Salesforce (Event) | On submission |
| New leads | VoiceLine → Salesforce | On submission |
| New accounts | VoiceLine → Salesforce | On submission |
| New contacts | VoiceLine → Salesforce | On submission |

## Authentication

Salesforce uses **OAuth 2.0**. The workspace administrator connects once via a browser-based authorization flow. VoiceLine stores the access token and refreshes it automatically — no credentials are stored directly.

## FAQ

**Do all reps need to authorize Salesforce individually?**
No — a single workspace-level OAuth connection covers all reps in the workspace. Individual reps do not need Salesforce credentials to use VoiceLine.

**Which Salesforce editions are supported?**
VoiceLine requires API access, which is available from Salesforce Professional edition and above.

**Can I push opportunities to Salesforce?**
Yes — reps can create and update opportunities via voice. VoiceLine syncs these to Salesforce including stage, amount, close date, and linked account. See [Opportunity](/customer/productivity/workflows/opportunity).
