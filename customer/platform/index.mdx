Workspace configuration — users, integrations, data, and system settings.

Workspace Settings covers everything a workspace administrator manages: connecting VoiceLine to your CRM, configuring what data is available to reps, and setting up language, security, and search behavior. Most of these settings are configured once during implementation and rarely changed.

## In this section

| Section | What it covers |
|---------|---------------|
| [Workspace Administration](/customer/platform/workspace-admin) | Users, roles, invitations, and workspace name |
| [Language Settings](/customer/platform/language-settings) | Input language, output language, custom vocabulary |
| [Security & Compliance](/customer/platform/security) | Authentication methods (password, OAuth, SSO) and data processing consent |
| [Workflow Configuration](/customer/platform/workflow-customisation) | Configurable field schemas for each workflow type |
| [Search Settings](/customer/platform/search-settings) | Search behavior, favourites, ownership filters, and permissions |
| [Integrations](/customer/platform/integrations/index) | CRM connections: SAP C4C, Salesforce, MS Dynamics, Aurea, ZOHO, HubSpot |
| [Data Objects](/customer/platform/data-objects/index) | Master records — accounts, contacts, products, competitors, and more |
| Unmatched Terms | Review product terms from recordings that could not be matched automatically and link them to the correct product or group |

## Typical setup order

1. **Integrations** — connect your CRM so account and contact data flows in
2. **Data Objects** — verify products and competitors are loaded
3. **Workspace Administration** — invite users and set roles
4. **Language Settings** — configure recording language and output language
5. **Search Settings** — configure ownership filters and extended search if needed
6. **Workflow Configuration** — customize schemas for your team's workflow if needed

## Who manages this

All settings in this section require the **workspace admin** role. Individual reps have no access to Workspace Settings.
