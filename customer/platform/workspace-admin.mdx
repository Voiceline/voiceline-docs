Manage your VoiceLine workspace — users, roles, and workspace-wide settings.

## Overview

Workspace Administration is where workspace administrators manage everything that affects all users: the workspace name, team members, roles, and access permissions. Only users with the workspace admin role can access these settings.

## Where to find it

Go to **Workspace Settings** → **Administration** (workspace name and plan) or **Workspace Settings** → **User Management** (members and roles).

## Workspace name

The workspace name appears in reports and admin views. To change it, go to **Administration** → **Workspace name** → enter the new name → **Save**.

## User management

**Inviting users:**
1. Go to **Workspace Settings** → **User Management**
2. Click **Invite member**
3. Enter the user's email address
4. The user receives an invitation email with a setup link

**User roles:**
| Role | Access level |
|------|-------------|
| **Member** | Record visits, create workflows, use the assistant |
| **Admin** | All member access + Workspace Settings |

Admins can promote members to admin or remove users from the workspace.

**User status:**
- **Active** — user has completed setup and can log in
- **Invited** — invitation sent, setup not yet completed

Admins can resend invitation emails or send a password reset link from the user management table.

## FAQ

**How many users can I add?**
User limits depend on your workspace plan. Contact your VoiceLine account manager for information on seat limits.

**Can I remove an admin's admin rights?**
Yes — workspace admins can remove the admin role from any other admin in the user management table.

**What happens when a user is removed?**
The user loses access to VoiceLine immediately. Their historical recordings and actions remain in the workspace.
